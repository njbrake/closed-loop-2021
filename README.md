# closed-loop-2021
Presentation for 3M | M*Modal Closed Loop 2021

This code is to support a node addon module, for extra speed!

# Quick Start

```{bash}
docker compose build
docker compose run cl bash
# Now we are inside docker container
yarn install
yarn start
```
